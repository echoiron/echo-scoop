
# echo-scoop

### How to install bucket

```bash
scoop bucket add echo https://github.com/echoiron/echo-scoop
```

### App of installing bucket

```bash
scoop install echo/app_name

# e.g.
scoop install echo/fscapture
```

### Delete all buckets
```bash
scoop bucket rm *
```
